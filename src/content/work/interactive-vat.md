Inspired by [Jorge Lescale](https://www.artstation.com/lescale) [demo](https://www.artstation.com/artwork/DAG2nn), I wanted to experiment with Vertex Animation Textures
 (VAT) and how they can help create interactivity in a game world.

I've used **Houdini** the create the simulations using Vellum and to export the textures, and used Unreal Engine 5 to prototype the interactivity. I believe this could still be further optimized, but it works well as a proof of concept.
