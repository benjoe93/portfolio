Holographic shader study made in Unreal Engine 5

I was inspired by **Star Wars Jedi: Fallen Order**'s map and Holo table and wanted to create a similar material from scratch. I used a few techniques that I haven't had the chance to try out before and I've also had the opportunity to use HLSL code in Unreal.

The original goal was to create the material, but when that was complete I couldn't stop and started experimenting with Niagara. This was my first attempt to create a particle system from scratch without any tutorials.

Models created by [Ruslan Malovsky](https://www.artstation.com/malovsky) and [Ashkelon](https://sketchfab.com/3d-models/lion-statue-660c1574ac8c4700b4c86b4376f40334)
